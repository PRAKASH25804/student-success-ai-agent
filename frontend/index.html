                        Documentation & Platform Architecture

                        System Architecture Overview

                        The Student Success AI Agent is structured as a Python Flask REST API backend connected to a SQLite database, delivering results to a responsive vanilla JavaScript Single Page Application.

                        [Frontend Vanilla HTML5/CSS/JS]
       │             ▲
       ▼             │  REST APIs (JSON)
[Flask Python Web Controller app.py]
  ├── db.py ──────────────► [SQLite Database (database.db)]
  ├── resume_analyzer.py ─► [Extract Resume text via PyPDF] ──┐
  ├── skill_gap.py ───────┐                                   │
  ├── interview_generator ├──► [Central Gemini API Client] ───┼─► [Gemini 2.5 Flash API]
  ├── study_planner.py ───┤                                   │
  └── roadmap_generator.py┘                                   ┘

                        Installation Instructions

                        1. Create Python virtual environment: python -m venv venv
                        2. Activate environment: venv\Scripts\activate (Windows)
                        3. Install dependencies: pip install -r requirements.txt
                        4. Create .env file in root and add: GEMINI_API_KEY=your_key
                        5. Run application: python backend/app.py
                        6. Open http://localhost:5000 in your browser.
